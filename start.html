Thank you for helping out the Melbourne police department.
With your expertise and detective skills we should be able to solve this important case.

We can’t afford any mistakes, so pay close attention before you get started.

OVERVIEW:
You’ll have 60 minutes to solve this case. If you haven’t solved it within 60 minutes it’s game over.
The time starts counting down as soon as you press Start Investigation.



Make sure you are all on a video/audio call in case you are not physically together.

TEAMWORK:
To be successful, you'll need to work closely together.
Before submitting an answer or requesting a hint, first discuss with the team.

Only 1 person in the team needs to submit an answer or request a hint.

After a few seconds everyone will see the result.

PENALTIES:
You can get help in case you’re stuck.
Press the Request Hint button and we’ll find a detective to help you out.

Requesting a hint results in a 30 second time penalty.

Submitting an incorrect answer results in a 30 second time penalty.
